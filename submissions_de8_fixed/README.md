# Đề số 08 - Bộ câu hỏi Python Lập trình Cơ bản

## Nội dung

Folder này chứa các giải pháp cho 10 bài tập Python:

### Câu 1: Fibonacci
- **Cau1a.py**: Tính số Fibonacci thứ n
- **Cau1b.py**: Tính Fibonacci với validation cho số âm

### Câu 2: Danh sách và Tần suất
- **Cau2a.py**: Sắp xếp danh sách bằng Bubble Sort
- **Cau2b.py**: Tìm phần tử xuất hiện nhiều nhất

### Câu 3: Xử lý chuỗi
- **Cau3a.py**: In các ký tự chung của hai chuỗi (sorted)
- **Cau3b.py**: In ký tự chỉ xuất hiện trong chuỗi thứ nhất (sorted)

### Câu 4: JSON Processing
- **Cau4a.py**: Đọc JSON và in tên người có tuổi lớn nhất
- **Cau4b.py**: Tính tuổi trung bình từ JSON

### Câu 5: Hình học (OOP)
- **Cau5a.py**: Tính diện tích hình tròn (base class HinhTru)
- **Cau5b.py**: Tính chu vi hình tròn

## Chạy chương trình chấm điểm

```bash
python autograde_de8.py
```
